# LGE-MRI Quality Assessment

This repository contains the code for training and evaluating the LGE-MRI quality assessment models.

## How to Run the Project

### 1. Activate the Conda Environment
Before running the code, activate your conda environment:

```bash
conda activate lge_mri_qc_multiclass
```
### 2. Run the bash file

```bash
sh run.sh
```
The Configuration of Experiment can be changed in the config.py File
